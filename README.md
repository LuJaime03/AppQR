# AppQR
## Generador de códigos QR con Python :)

Este proyecto es una herramienta con interfaz gráfica moderna (GUI) que permite crear archivos QR de forma rápida.

### Características principales:

**Interfaz Moderna:** Desarrollada con CustomTkinter para un diseño elegante y modo oscuro.

**Gestión de archivos:** Integración con filedialog para que el usuario elija la ubicación y nombre de guardado.

**Previsualización en tiempo real:** El código QR se muestra en la ventana apenas se genera.

**Manejo de errores:** Sistema de alertas para campos vacíos o errores de escritura.
